CacheCleaner

Type Plugin (shell)
Etat Stable
Compatibilité CakePHP 3.x
Description Ce plugin pour CakePHP 3.x permet de néttoyer le cache de votre application en production (très utile après une mise en production). Il gère le système de cache de fichiers (dossier tmp), le cache ORM, le cache applicatif (cakephp), ainsi que l'Opcache.
Dépendences Aucune
