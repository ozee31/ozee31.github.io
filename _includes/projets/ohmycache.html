OhMyCache

Type Librairie
Etat Stable
Description Basé sur les API LocalStorage et SessionStorage, cette librairie permet de stocker n'importe quel type de données (string, json, array...) et ajoute la possibilité de les enregistrer de manière temporaire et/ou en lecture seule.
Développement Grâce à Webpack, la librairie est côdée en ECMAScript 2015 (puis compilé grâce à Babel). De plus elle utilise le pattern UMD, ce qui la rend compatible AMD, CommonJS...
Dépendences Aucune
