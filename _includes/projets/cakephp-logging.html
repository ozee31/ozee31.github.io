Logging

Type Plugin
Etat Stable
Compatibilité CakePHP 3.x
Description Initialement conçu pour permettre la journalisation des actions des utilisateurs, ce plugin vous permet de stocker n'importe quels évenements dans votre base de données. Il est également accompagné d'un component qui permet de stocker à la demande (ou de manière automatique) des données comme l'objet Session ou l'objet Request
Dépendences Aucune
